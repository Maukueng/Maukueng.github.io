单调栈

单调栈的概念

单调栈就是一个栈结构，里面存放的内容从栈顶到栈底是依次递增或者递减的。它可以用来解决在一个数组中找出每个元素对应左右两部分比自己小的值并且最近的值的情况。并且保证时间复杂度在O(n)。

单调栈的解题模板

stack<int> st; //根据需求，单调增或减
//一般寻找下一个更大元素的时候从后往前遍历,且栈顶到栈底递增
for (遍历数组) {
	while (栈非空 && 当前元素大于等于栈顶元素) {
		弹出栈顶元素
	}
	if (栈非空) {
		栈顶元素就是下一个更大的元素
	}
	当前元素进栈
}

单调栈的应用

739. 每日温度 - 力扣（LeetCode）

给定一个整数数组 temperatures ，表示每天的温度，返回一个数组 answer ，其中 answer[i] 是指对于第 i 天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，请在该位置用 0 来代替。

示例 1:

输入: temperatures = [73,74,75,71,69,72,76,73] 输出: [1,1,4,2,1,1,0,0]

代码：

vector<int> dailyTemperatures(vector<int>& temperatures) {
        //单调栈，栈顶到栈底递增
        stack<int> st; //存放下标
        int n = temperatures.size();
        //存放答案，设置初值
        vector<int> res(n, 0);
        for (int i = n - 1; i >= 0; --i) {
        	//若当前温度大于等于栈顶元素，则弹出栈顶元素
            while (!st.empty() && temperatures[i] >= temperatures[st.top()]) {
                st.pop();
            }
            if (!st.empty()) {
                res[i] = st.top() - i;
            }
            st.push(i);
        }
        return res;
    }
